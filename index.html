入门 React Redux

官方维护

React Redux 由 Redux 官方团队维护，并 与 Redux 和 React 最新的 API 保持同步。

可预测

旨在与 React 的组件模型一起使用。你来定义如何从 Redux 中提取组件所需的数据，然后组件即可根据需要自动更新。

封装

提供开箱即用的 API 实现组件与 Redux Store 交互，避免手动编写代码实现逻辑。

高性能

内部自动做了复杂的性能优化，帮助组件仅在它需要的数据变化后才重新渲染。
